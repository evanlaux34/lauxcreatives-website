Square, letterpress-feel call-to-action with an uppercase Courier Prime label — the brand's primary button.

```jsx
<Button variant="primary" size="md">Inquire</Button>
<Button variant="secondary">View Portfolio</Button>
<Button variant="ghost">Read more</Button>
<Button variant="onDark" as="a" href="#">Get in touch</Button>
```

Variants: `primary` (Sepia Ink fill), `secondary` (outline), `ghost` (text-only, warms to gold), `onDark` (for photo/dark surfaces). Sizes `sm | md | lg`. Square corners, soft hover lift, restrained press (1px nudge) — no bounce.
