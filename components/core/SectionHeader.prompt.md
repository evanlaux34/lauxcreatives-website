The brand's section header — a numbered eyebrow paired with a right-aligned meta label over a hairline rule.

```jsx
<SectionHeader number="02" title="Palette" meta="Pulled from film" />
<SectionHeader number="04" title="The Work" meta="Weddings · Engagements" tone="onDark" />
```

Set `tone="onDark"` on dark/photo sections. `rule={false}` drops the hairline.
